# Enkel og forståelig browsertesting

## Hva er dette?

Dette er en liten demo-app som forklarer hvordan du kan skrive din egen test-runner og sette den opp for å kjøre automatiserte tester i nettleseren. Du kan bruke dette til å skrive og kjøre enhetstester og grensesnittester uten å installere et eksternt rammeverk. Du kan bruke konseptene her i din egen kode, men vær obs på at dette er ment som et proof of concept og kanskje ikke egner seg for større og mer kompliserte produksjonsmiljøer.

## Komme i gang

Du må ha Node.js og npm for å kjøre dette prosjektet fra terminalen. Du kan også bruke VS Code sin liveserver om du heller vil det; da må du hoste den fra `/app`-mappa.

+ Klon repoet til din maskin.
+ Naviger deg til mappa i terminalen og kjør `npm install` for å installere avhengigheter.
+ Start prosjektet med `npm start` og naviger deg til [localhost:12345](http://localhost:12345) for å se nettsiden.
